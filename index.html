HTML

  • Hypertext Markup Language (HTML) is used to give the content structure.
  • The head element contains information about the webpage.
  • The title element defines the title for a webpage, which is required and important for SEO.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The body element represents the visible content shown to the user.
  • The h1, h2, h3, etc elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text.
  • The ul, ol and li elements represent unordered lists, ordered lists and list items.
  • The img element contains info about images that are displayed on the webpage.
  • The br element creates a line of empty space between two blocks of content.
  • The a, anchor, element creates links to the same or other webpages.

CSS

  • Cascading Style Sheets is used to define how webpage content is presented to users.
  • Inline CSS is used to style an element directly in the HTML file; good for quick and permanent changes.
  • Internal style sheets contain CSS rules for the webpage in the head section of the HTML file.
  • External style sheets are preferred because it allows all CSS rules to exist in a separate file, making design changes easier.
  • Selector defines the element or attributes to which the rules will apply.
  • Declarations contain 2 components: the CSS property we want to apply and the value of the property.
  • A margin property indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside an element.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule with a class selector.
